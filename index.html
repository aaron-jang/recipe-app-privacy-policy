🍽️ 나만의 레시피 앱 개인정보처리방침

요약: 본 앱은 사용자의 개인정보를 수집하거나 저장하지 않습니다. 모든 데이터는 사용자의 기기에만 저장되며, 외부로 전송되지 않습니다.

1. 수집하는 개인정보

  • 본 앱은 개인정보를 수집하지 않습니다
  • 모든 레시피 데이터는 사용자 기기에만 저장됩니다
  • 사용자 식별 정보, 연락처, 위치 정보 등을 수집하지 않습니다

2. 권한 사용 목적

  • 카메라 권한 (android.permission.CAMERA)
    • 목적: 레시피 사진을 직접 촬영하기 위해서만 사용
    • 저장: 촬영된 사진은 사용자 기기에만 저장
    • 전송: 촬영된 사진을 외부로 전송하지 않음
  • 저장소 권한
    • 목적: 레시피 사진을 기기에 저장하고 불러오기 위해서만 사용
    • 범위: 앱이 생성한 파일에만 접근
  • 인터넷 권한
    • 목적: YouTube 영상 정보 가져오기 및 광고 표시
    • 개인정보: 사용자 레시피 데이터는 전송하지 않음

3. 데이터 보관 및 처리

  • 모든 레시피 데이터는 사용자 기기에만 저장됩니다
  • 외부 서버나 클라우드로 데이터를 전송하지 않습니다
  • 사용자가 앱을 삭제하면 모든 데이터가 함께 삭제됩니다
  • 앱 업데이트 시에도 기존 데이터는 보존됩니다

4. 제3자 서비스

  • Google AdMob
    • 광고 표시를 위해 사용됩니다
    • 광고 식별자(Advertising ID)를 사용할 수 있습니다
    • 사용자 레시피 데이터에는 접근하지 않습니다
  • YouTube API
    • YouTube 영상 정보 가져오기에만 사용됩니다
    • 사용자 개인정보를 YouTube에 전송하지 않습니다

5. 데이터 보안

  • 모든 데이터는 사용자 기기에 암호화되어 저장됩니다
  • 네트워크를 통한 개인정보 전송이 없어 보안 위험이 최소화됩니다
  • 사용자가 직접 데이터를 관리하고 제어할 수 있습니다

6. 사용자 권리

  • 언제든지 앱 설정에서 권한을 철회할 수 있습니다
  • 앱 삭제를 통해 모든 데이터를 완전히 제거할 수 있습니다
  • 개인정보처리방침 관련 문의는 언제든지 가능합니다

7. 아동의 개인정보보호

  • 본 앱은 13세 미만 아동을 대상으로 하지 않습니다
  • 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다

8. 개인정보처리방침 변경

  • 개인정보처리방침 변경 시 앱 업데이트를 통해 공지됩니다
  • 중요한 변경사항은 별도 알림을 제공합니다
