🍽️ 나만의 레시피 앱 개인정보처리방침

요약: 본 앱은 사용자의 개인정보를 수집하거나 저장하지 않습니다. 모든 데이터는 사용자의 기기에만 저장되며, 외부로 전송되지 않습니다.

1. 수집하는 개인정보

  • 본 앱은 개인정보를 수집하지 않습니다
  • 모든 레시피 데이터는 사용자 기기에만 저장됩니다
  • 사용자 식별 정보, 연락처, 위치 정보 등을 수집하지 않습니다

2. 권한 사용 목적

  • 카메라 권한 (android.permission.CAMERA)
    • 목적: 레시피 사진을 직접 촬영하기 위해서만 사용
    • 저장: 촬영된 사진은 사용자 기기에만 저장
    • 전송: 촬영된 사진을 외부로 전송하지 않음
  • 저장소 권한
    • 목적: 레시피 사진을 기기에 저장하고 불러오기 위해서만 사용
    • 범위: 앱이 생성한 파일에만 접근
  • 인터넷 권한
    • 목적: YouTube 영상 정보 가져오기 및 광고 표시
    • 개인정보: 사용자 레시피 데이터는 전송하지 않음

3. 데이터 보관 및 처리

  • 모든 레시피 데이터는 사용자 기기에만 저장됩니다
  • 외부 서버나 클라우드로 데이터를 전송하지 않습니다
  • 사용자가 앱을 삭제하면 모든 데이터가 함께 삭제됩니다
  • 앱 업데이트 시에도 기존 데이터는 보존됩니다

4. 제3자 서비스

  • Google AdMob
    • 광고 표시를 위해 사용됩니다
    • 광고 식별자(Advertising ID)를 사용할 수 있습니다
    • 사용자 레시피 데이터에는 접근하지 않습니다
  • YouTube API
    • YouTube 영상 정보 가져오기에만 사용됩니다
    • 사용자 개인정보를 YouTube에 전송하지 않습니다

5. 데이터 보안

  • 모든 데이터는 사용자 기기에 암호화되어 저장됩니다
  • 네트워크를 통한 개인정보 전송이 없어 보안 위험이 최소화됩니다
  • 사용자가 직접 데이터를 관리하고 제어할 수 있습니다

6. 사용자 권리

  • 언제든지 앱 설정에서 권한을 철회할 수 있습니다
  • 앱 삭제를 통해 모든 데이터를 완전히 제거할 수 있습니다
  • 개인정보처리방침 관련 문의는 언제든지 가능합니다

7. 아동의 개인정보보호

🍽️ My Recipe App Privacy Policy

Summary: This app does not collect or store any personal information from users. All data is stored only on the user's device and is not transmitted externally.

1. Personal Information Collection

  • This app does not collect personal information
  • All recipe data is stored only on the user's device
  • We do not collect user identification information, contacts, location data, etc.

2. Permission Usage

  • Camera Permission (android.permission.CAMERA)
    • Purpose: Used only for taking recipe photos directly
    • Storage: Photos are stored only on the user's device
    • Transmission: Photos are not transmitted externally
  • Storage Permission
    • Purpose: Used only for saving and loading recipe photos on the device
    • Scope: Access only to files created by the app
  • Internet Permission
    • Purpose: Retrieving YouTube video information and displaying ads
    • Privacy: User recipe data is not transmitted

3. Data Storage and Processing

  • All recipe data is stored only on the user's device
  • Data is not transmitted to external servers or cloud services
  • All data is deleted when the user uninstalls the app
  • Existing data is preserved during app updates

4. Third-Party Services

  • Google AdMob
    • Used for displaying advertisements
    • May use Advertising ID
    • Does not access user recipe data
  • YouTube API
    • Used only for retrieving YouTube video information
    • Does not transmit user personal information to YouTube

5. Data Security

  • All data is stored encrypted on the user's device
  • Security risks are minimized as no personal information is transmitted over the network
  • Users can directly manage and control their data

6. User Rights

  • Users can revoke permissions at any time in app settings
  • All data can be completely removed by uninstalling the app
  • Inquiries about the privacy policy are always welcome

7. Children's Privacy Protection

  • This app is not intended for children under 13
  • We do not intentionally collect personal information from children under 13

8. Privacy Policy Changes

  • Changes to the privacy policy will be announced through app updates
  • Important changes will be provided with separate notifications

9. Contact Information

  • App Name: My Recipe (나만의 레시피)
  • Developer: Recipe App Developer
  • Email: soosoolife.k@gmail.com
  • Response Time: Within 48 hours on business days

🍽️ マイレシピアプリ プライバシーポリシー

概要: 本アプリはユーザーの個人情報を収集・保存いたしません。すべてのデータはユーザーの端末にのみ保存され、外部に送信されることはありません。

1. 収集する個人情報

  • 本アプリは個人情報を収集いたしません
  • すべてのレシピデータはユーザーの端末にのみ保存されます
  • ユーザー識別情報、連絡先、位置情報等は収集いたしません

2. 権限の使用目的

  • カメラ権限 (android.permission.CAMERA)
    • 目的: レシピの写真を直接撮影するためのみに使用
    • 保存: 撮影された写真はユーザーの端末にのみ保存
    • 送信: 撮影された写真を外部に送信いたしません
  • ストレージ権限
    • 目的: レシピの写真を端末に保存・読み込みするためのみに使用
    • 範囲: アプリが作成したファイルにのみアクセス
  • インターネット権限
    • 目的: YouTube動画情報の取得および広告表示
    • 個人情報: ユーザーのレシピデータは送信いたしません

3. データの保管及び処理

  • すべてのレシピデータはユーザーの端末にのみ保存されます
  • 外部サーバーやクラウドにデータを送信いたしません
  • ユーザーがアプリを削除すると、すべてのデータも一緒に削除されます
  • アプリ更新時も既存データは保持されます

4. 第三者サービス

  • Google AdMob
    • 広告表示のために使用されます
    • 広告識別子（Advertising ID）を使用する場合があります
    • ユーザーのレシピデータにはアクセスいたしません
  • YouTube API
    • YouTube動画情報の取得のみに使用されます
    • ユーザーの個人情報をYouTubeに送信いたしません

5. データセキュリティ

  • すべてのデータはユーザーの端末に暗号化されて保存されます
  • ネットワークを通じた個人情報送信がないため、セキュリティリスクが最小化されます
  • ユーザーが直接データを管理・制御できます

6. ユーザーの権利

  • いつでもアプリ設定で権限を取り消すことができます
  • アプリ削除によりすべてのデータを完全に削除できます
  • プライバシーポリシーに関するお問い合わせはいつでも可能です

7. 児童の個人情報保護

  • 本アプリは13歳未満の児童を対象としておりません
  • 13歳未満の児童の個人情報を意図的に収集いたしません

8. プライバシーポリシーの変更

  • プライバシーポリシー変更時はアプリ更新を通じてお知らせいたします
  • 重要な変更事項については別途通知を提供いたします
